# Poema do Git

No começo era o caos, sem versão,
Cada arquivo perdido na escuridão.
Veio o Git, com seu commit certeiro,
Salvando o código do programador brasileiro.

Branch que nasce, branch que floresce,
Merge que junta o que o tempo esquece.
Push pra nuvem, pull (pum)? pra voltar,
E o Rapha aprende a colaborar.

Conflito de merge? Vem cá, meu amigo,
Resolve com calma, sem stress antigo.
Que o Git é assim, ferramenta de irmão:
Une o que separa, guarda a evolução.
